# 品質レビューチェックリスト

## レビュー方針

chrome-devtools MCPを使用し、**サンプリング方式**でスクリーンショットベースのレビューを行う。

### サンプリング対象

全スライドを撮影するのではなく、以下を選択する：
1. カバースライド（1枚目）
2. セクション区切り（1枚）
3. 使用した各familyから代表1枚ずつ
4. 合計5〜7枚程度

同じfamilyはテーマトークンが同じため、1枚見れば品質がわかる。問題発見時のみ全ページレビューに切り替える。

### レビュー手順

```
1. navigate_page でHTMLファイルを開く
2. resize_page で 1920x1080 に設定
3. evaluate_script で navigateToSlide(N) を実行
4. take_screenshot でスクリーンショット撮影
5. チェックリストに基づいてレビュー
6. 問題があればHTML修正 → 該当スライドのみ再撮影
```

## チェック項目

### 1. レイアウト

- [ ] テキストがスライド領域（1920x1080）をはみ出していないか
- [ ] 各要素間の余白が十分にあるか（詰め込みすぎていないか）
- [ ] **余白が過剰でないか**（コンテンツが上部に寄り、下半分が空いていないか）
  - コンテンツ量が少ない場合: 要素を大きくする / 説明文を追加する / 垂直中央に配置する
- [ ] 左右のパディングが均等か（最低80px = 5rem）
- [ ] 上下のパディングが均等か（ヘッダー下に十分なマージン）
- [ ] 要素の水平・垂直アラインメントが揃っているか
- [ ] グリッドレイアウトでカードの高さが揃っているか
- [ ] **コンテンツ幅に `max-w-*` を使う場合、左右の余白が均等か**（`mx-auto` で中央配置すること）

### 2. タイポグラフィ

- [ ] フォントサイズの階層が一貫しているか
  - 見出し: 48-52px
  - サブ見出し: 28-36px
  - 本文: 22-26px
  - 補足: 16-20px
- [ ] 見出しが全スライドで同じサイズ/ウェイトか
- [ ] 行間（line-height）が適切か（本文は1.6〜1.8推奨）
- [ ] テキストが途中で切れていないか
- [ ] 日本語フォント（Noto Sans JP）が正しく読み込まれているか

### 3. テーマ一貫性

- [ ] テーマカラーが全スライドで統一されているか
- [ ] アクセントカラーが見出しに正しく適用されているか
- [ ] 背景色がfamily/variantに応じて正しいか
  - コンテンツスライド: `--color-bg`（白）
  - section-divider: グラデーションまたは `--color-primary`
  - カード背景: `--color-bg-alt`
- [ ] ボーダー、区切り線のスタイルが統一されているか
- [ ] **同一スライド内の並列要素のデザインが統一されているか**
  - 例: 2枚のカードの一方にだけバッジがある、ボーダー色が異なる等は不可
  - 並列要素は同じ構造（バッジあり/なし、ボーダー色、パディング）で揃える

### 4. 画像

- [ ] 画像のアスペクト比が崩れていないか（`object-cover` or `object-contain`）
- [ ] プレースホルダー画像のサイズが適切か
- [ ] 画像がスライド外にはみ出していないか
- [ ] team-membersの丸写真が正円になっているか

### 5. 情報量

- [ ] 1スライドの情報量が適切か（「1スライド1メッセージ」原則）
- [ ] 箇条書きが5項目以内か
- [ ] テキストの文字数が多すぎないか（本文は1段落3行以内推奨）
- [ ] 図やダイアグラムの要素数が多すぎないか

### 6. コントラスト・可読性

- [ ] テキストと背景のコントラスト比が十分か（WCAG AA: 4.5:1以上）
- [ ] 薄い色のテキストが読みにくくないか
- [ ] section-divider上の白テキストが背景色に対して読めるか

### 7. ナビゲーション

- [ ] プログレスバーが表示されているか
- [ ] スライド番号が正しく表示されているか
- [ ] キーボード操作（←→）で正しく移動するか
- [ ] 最初/最後のスライドでエラーが出ないか

### 8. フッター・ヘッダー

- [ ] ロゴが全コンテンツスライドの左上に表示されているか
- [ ] ページ番号がフッターに正しく表示されているか
- [ ] セクション名がフッターに表示されているか
- [ ] コピーライトが右下に表示されているか

## 自動修正の優先順位

問題が見つかった場合、以下の優先順位で修正する：

1. **Critical**: テキストはみ出し、要素の重なり、表示崩れ
2. **High**: テーマカラーの不統一、フォントサイズの不整合
3. **Medium**: 余白の過不足、アラインメントのずれ
4. **Low**: 情報量の最適化、装飾的な改善
